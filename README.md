# 🚀 QUANT-AI

An AI-powered data analysis and prediction system that allows users to upload datasets and get insights, visualizations, and predictions — without coding.

---

## 🔥 Features

- 📊 Upload datasets (CSV)
- 📈 Automatic data analysis & visualization
- 🤖 Machine Learning predictions
- ⏳ Time series forecasting
- 💬 Sentiment analysis

---

## 🛠 Tech Stack

- **Backend:** FastAPI
- **Language:** Python
- **ML Libraries:** Pandas, Scikit-learn
- **Frontend:** HTML, CSS
